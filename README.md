### _Igniting your curiosity with every spark_

## Description
This is a web app for sparking conversations, posting questions and answers.

## Funtionality

- Register and login validation with JWT
- Protected routes prevents unregistered users from accessing the site
- User can post and answer questions, filter them and sort by date posted
- User can edit and delete their own posts
